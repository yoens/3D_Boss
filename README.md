## 2. 구현 기능

### 플레이어
- 마우스 조준 회전 — 지면 Raycast로 커서 위치를 향해 회전, 이동 방향과 바라보는 방향이 독립
- 회피(백스텝) + 무적 프레임(i-frame)
- 공격 — 애니메이션 이벤트 기반 판정
- 2D 블렌드 트리 기반 8방향 이동 애니메이션

### 몬스터 AI
- 클래스 기반 유한 상태 기계(FSM) — Idle / Chase / Attack / Dead
- 추격, 공격 범위 판정, 공격 쿨타임
- 콤보 공격 — 트리거 1회로 3연타 진행, 타격마다 개별 판정
- 피격 시 연계 중단(경직)

### 공통
- `IDamageable` 인터페이스 기반 피격 처리
